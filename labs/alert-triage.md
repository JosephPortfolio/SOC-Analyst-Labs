# 🛑 Room Walkthrough: SOC L1 Alert Triage

## 🎯 Objective
Practice a systematic approach to analyzing incoming security alerts, minimizing alert fatigue, and distinguishing true positives from false positives.

## 🛠️ Skills & Concepts Learned
* **Indicator Analysis:** Identifying malicious source IPs and system process anomalies.
* **Triage Workflow:** Utilizing playbooks to prioritize alerts based on threat severity.
* **Remediation:** Documenting the immediate steps required to contain a verified threat.

## 🚀 Key Takeaways
* Handled alert validation pipelines efficiently to lower Mean Time to Acknowledge (MTTA).
* Analyzed process lineage to find true malicious indicators.
